Fundamentos Essenciais





Conheça mais um pouco sobre as bases indispensáveis da criação web, programação e design digital.

Os fundamentos essenciais são o alicerce para qualquer desenvolvedor. Compreender as bases da tecnologia e do design digital é crucial para o sucesso em projetos futuros.


HTML




HTML (HyperText Markup Language) é a linguagem básica usada para criar páginas na web. Utiliza tags para estruturar e definir conteúdo, como parágrafos (<p>), cabeçalhos (<h1> a <h6>), links (<a>) e imagens (<img>). É essencial para a criação da estrutura de sites.

Este foi o meu primeiro contato com programação de sites. Gostei bastante de aprender, pois foi uma experiência muito divertida com professores excelentes e muito legais.


CSS




CSS (Cascading Style Sheets) é usado para estilizar páginas web, controlando cores, fontes, espaçamentos e layouts. Funciona através de regras que aplicam estilos a elementos HTML, permitindo criar designs visuais atraentes e layouts responsivos para diferentes dispositivos.

Nesse ponto, já estava familiarizado com HTML, e começamos a combinar os dois para criar sites mais bonitos e elegantes. Gostei bastante de trabalhar com HTML e CSS juntos.


Python




Python é uma linguagem de programação de alto nível conhecida por sua simplicidade e versatilidade. É usada em várias áreas como desenvolvimento web, automação, análise de dados e inteligência artificial. Sua sintaxe clara e uma vasta biblioteca tornam-na ideal tanto para iniciantes quanto para projetos complexos.

Aqui, aprendi os conceitos básicos de Python e suas aplicações práticas, o que ampliou minha compreensão sobre programação e suas possibilidades.


PyCharm




PyCharm é um IDE desenvolvido pela JetBrains para Python. Oferece ferramentas como autocompletar código, depuração, testes unitários e integração com controle de versão. Suporta frameworks web como Django e Flask, ajudando desenvolvedores a escrever código eficiente e de alta qualidade.

Descobri que o PyCharm é uma extensão para Python. Embora não tenha achado tão intuitivo no início, reconheço seu valor como ferramenta avançada para desenvolvedores.


TinkerCAD




TinkerCAD é uma ferramenta online gratuita da Autodesk para design 3D, eletrônica e codificação. Ideal para iniciantes, permite criar modelos 3D usando formas básicas, simular circuitos eletrônicos e aprender programação com blocos visuais ou Arduino, tornando-se uma ferramenta educacional valiosa.

No TinkerCAD, gostei de aprender a fazer um semáforo, compreendendo tanto a sua programação quanto o funcionamento físico. Foi uma experiência muito enriquecedora.


Aula Técnica de Demonstração de Gabinete




Essa aula ensina sobre o funcionamento e as características dos gabinetes de computador. Sobre tipos de gabinetes, ventilação e refrigeração, organização de cabos, compatibilidade com componentes, e como configurar e otimizar gabinetes para diferentes usos, como jogos, design gráfico e servidores.

Foi uma aula bastante interessante, onde aprendi sobre todas as peças dentro do Gabinete, como montá-las e suas respectivas funções.


Figma: Ferramenta Poderosa para Design de Interface de Usuário




Figma é uma ferramenta de design colaborativo baseada em nuvem, usada para criar interfaces e experiências de usuário. Lançado em 2015, permite que designers trabalhem juntos em tempo real, facilitando a colaboração e o feedback instantâneo. Suas funcionalidades incluem prototipagem, design responsivo e componentes reutilizáveis, acessíveis diretamente do navegador. Figma é essencial para equipes de design modernas, integrando fluxos de trabalho de design e desenvolvimento de maneira eficiente.

O Figma foi uma ferramenta excepcional para mim, pois aprendi a organizar e planejar um site antes de sua implementação final em outras plataformas.
